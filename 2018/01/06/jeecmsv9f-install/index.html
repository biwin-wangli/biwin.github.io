简介

JEECMSv9源码工程转为Maven版本，后台页面采用VUE技术全面进行了改版，新版本更注重用户体验，针对内容管理、留言、评论等功能进行了操作优化，对系统在海量数据读写性能上进行了大量的优化。

V9版本具备的特点：

  • 全站静态化解决方案
  • 网站前后台分离部署解决方案
  • 微信小程序解决方案
  • 站群统一管理解决方案
  • 可视化模板制作
  • 完善的API标准接口
  • 提供全部源码
  • 信息采集爬虫解决方案
  • 自定义栏目模型解决方案
  • 微信插件解决方案
  • 一站多终端解决方案
  • 支持集群部署或分布式部署
  • 自定义工作流
  • 易于二次开发
  • 数据报表解决方案
  • 站内信解决方案
  • 网站防火墙解决方案
  • 站群内容共享推送解决方案
  • 内容权限控制解决方案
  • 内容电商方案，提供收费和打赏模式
  • 自定义内容模型解决方案
  • 定时任务解决方案
  • 插件化模块管理解决方案
  • 类百度文库,可手机浏览器阅读
  • 数据备份恢复解决方案
  • 跨栏目内容共享解决方案
  • 静态分离式发布解决方案
  • 会员注册模型自定义解决方案
  • 多样化资源存储方案
  • 模板资源在线管理解决方案

运行环境

  • WEB服务器：Weblogic、Tomcat、WebSphere、JBoss 等
  • 数据库服务器：Oracle、DB2、Mysql、SqlServer、Sybase 等
  • 操作系统 ：Windows、Linux、Unix 等

本机源码搭建环境
IDE:

  • 后台src源码由MyEclipse2014直接由“已存在的Maven项目”的方式导入。
  • 后台管理界面是由WebStorm直接导入。

后台导入源码后的修改步骤：
1.在mysql客户端中直接源码DB文件夹中的建库脚本。
2.启动即可，此时需要记住tomcat的端口。

后台Vue管理导入IDE后的修改步骤：
1.在代码的跟目录中执行npm install，也可以用cnpm install,后者需要配置淘宝的npm。
2.修改config/dev.env.js中的baseUrl地址为后台源码地址
3.在代码的根目录中执行npm run start，启动好后，直接访问localhost:8889即可。
4.用户名为admin，密码为password。
