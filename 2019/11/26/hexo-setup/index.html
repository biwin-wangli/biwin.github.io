环境准备

  • 安装Nodejs（必须）

    作用：用来生成静态页面的 到Node.js官网下载相应平台的最新版本，一路安装即可。

  • 安装Git（必须）

    作用：哈哈， 这都不懂？好吧，还是简单说下，就是提交代码的工具而已。

  • 注册Gitee账号。（必须）

    作用：是用来做博客的代码仓库、并用它来展示我们的博客，就不用单独在申请域名了。

搭建博客

先在Gitee个人中心右上角点击“+”号，创建一个仓库，名字、路径都可以随意，不过我建议与个人空间地址的名字一致。

启用Gitee提供的免费域名

在项目首页 -> 服务 -> Gitee Pages

第一次用的用户，需要先绑定手机号，绑定成功后，选择对应的分支，点击部署，Gitee将生成地址。
